Introduction : My sites name is geeked and is essentially a newsletter for Technology ,Comics and Space enthusiasts.This sites searches for the latest and the greatest news from around the web and presents to its users in an aggregated manner .This helps users save time since they don't heed to visit/subscribe to multiple newsletters from different websites.

Structure of my website is

Common structure between all the pages :

  1. All the webpages have a header and a navigation bar for users to easily navigate between pages.
  2. All the webpages have font adjuster button at the header for helping users change the font 
  3. All the webpages have a footer which as a subscription button so users can easily subscribe


Homepage : This is the main page where the users can see all the main news/content from the world of Technology ,Comics and Space .

below is the page layout

  1. Left side contains a sidebar which auto scrolls the latest Technology news
  2.  Top portion contains a image sideshow which shows only one top news from each space ,technology and Comics category 
  3.  Below the sideshow contains the article cards for space news and comic news .They are divided by topic dividers


Space Page : Space page contains the below :

  1. A auto sideshow of latest Space news
  2. Article cards showing the different space news


Comics Page : Comics page contains the below :

  1. A auto sideshow of latest Comic news
  2. Article cards showing the latest comic book release
  3. A short video on one of the comic book release
  4. Article cards showing the latest comic news


Contact Page : A forum page where users can submit any messages


About us Page : List of all the employee and their social media info is provided in this page

Accessibility : Below are the three ways in which my site is accessible to those with different abilities and needs :

  1. All pages have font adjustor button on the left side of the header so that users can change the font very easily 
  2. alt tags with meaningful description have been used throughout the website for images
  3.  Sufficient Contrast has also been provided especially on text which are on top of images and all hyperlink have been colored with seperate blue color .

Usability : Below are the three ways i considered the usability of my site :

  1. Use of header and navigation bar in every page makes the website really easy to navigate
  2. Since all the contents are organized in article boxes its very easy for users to read
  3. Use of hamburger menu for mobile view will help users navigate the website in their cell phone

Learning : Below are the things i had to learn to create my site :

  1. Use of media tags : I had to learn how to use media tags  effectively
  2. Using Grid layout : I learned how to use grid view layout to organize the page
  3. Creating custom javascript functions :I learned how to create javascript functions to validate the forum and also to do dom manipulation
